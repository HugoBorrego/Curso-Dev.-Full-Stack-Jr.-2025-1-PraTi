Pipoca Doce Caramelada

  1. Em uma panela de fundo grosso, adicione todos os ingredientes.
  2. Misture bem e tampe a panela.
  3. Leve ao fogo médio e vá mexendo a panela com cuidado (use luvas ou panos para não se queimar).
  4. Quando os estouros começarem a diminuir, desligue o fogo imediatamente.
  5. Transfira a pipoca para uma travessa e mexa com uma colher para soltar os grãos caramelizados.
